💰 Conversor de Moedas 
Este é um Dashboard moderno e responsivo para conversão de moedas e acompanhamento de taxas de câmbio. O projeto foi desenvolvido com foco em experiência do usuário, oferecendo uma interface limpa e funcionalidades úteis para o dia a dia financeiro.

🚀 Funcionalidades
Conversão em Tempo Real (Simulada): Cálculo instantâneo entre diversas moedas globais.

Sistema Multilingue: Suporte completo para Português (BR), Inglês (US) e Espanhol (ES).

Modo Escuro (Dark Mode): Interface adaptável para maior conforto visual.

Tabela de Cotações: Visualização rápida do valor das principais moedas em relação ao Real.

Modo Privacidade: Opção para ocultar valores sensíveis na tela.

Design Responsivo: Menu lateral (Sidebar) adaptado para dispositivos móveis com navegação fluida.

🛠️ Tecnologias Utilizadas
React.js: Biblioteca principal para construção da interface baseada em componentes.
JavaScript (ES6+): Lógica de conversão e manipulação de estados.
CSS3: Estilização personalizada com Flexbox e Media Queries para responsividade.
FontAwesome: Ícones profissionais para navegação e interface.
Vite: Ferramenta de build para um desenvolvimento rápido e otimizado.

Conversor Multimoedas: Um formulário interativo onde o usuário pode selecionar moedas de origem e destino (como BRL para USD) e calcular valores com taxas pré-definidas.
Placar das Moedas (Tabela): Uma tabela comparativa que mostra quanto vale 1 unidade das principais moedas mundiais (Dólar, Euro, Libra, etc.) em relação ao Real (BRL).
Sistema de Temas: Alternância entre Modo Claro e Modo Escuro através de um ícone de lua/sol no cabeçalho.
Multilingue: Suporte completo para três idiomas: Português, Inglês e Espanhol, alterando todos os textos da interface instantaneamente.
Notícias do Mercado: Uma seção que exibe cards com as últimas movimentações do câmbio e links externos para sites de economia.
Modo Privacidade: Uma função nas configurações que permite ocultar os valores convertidos (exibindo "••••••") para maior discrição.

💰 Multi-Currency Converter
This is a modern and responsive Dashboard for currency conversion and exchange rate monitoring. The project was developed with a focus on user experience, offering a clean interface and useful features for daily financial activities.

🚀 Features
Real-Time Conversion (Simulated): Instant calculation between various global currencies.

Multilingual System: Full support for Portuguese (BR), English (US), and Spanish (ES).

Dark Mode: Adaptive interface for better visual comfort.

Exchange Rate Table: Quick view of the value of major currencies relative to the Brazilian Real (BRL).

Privacy Mode: An option to hide sensitive values on the screen.

Responsive Design: Side menu (Sidebar) adapted for mobile devices with fluid navigation.

🛠️ Technologies Used
React.js: Main library for building the component-based interface.
JavaScript (ES6+): Conversion logic and state management.
CSS3: Custom styling with Flexbox and Media Queries for responsiveness.
FontAwesome: Professional icons for navigation and UI.
Vite: Build tool for fast and optimized development.

🔍 Detailed Overview
Currency Converter: An interactive form where the user can select source and destination currencies (such as BRL to USD) and calculate values using pre-defined rates.
Currency Scoreboard (Table): A comparative table showing the value of 1 unit of the world's main currencies (Dollar, Euro, Pound, etc.) in relation to the Real (BRL).
Theme System: Toggle between Light Mode and Dark Mode via a moon/sun icon in the header.
Multilingual: Instant interface text updates for Portuguese, English, and Spanish.
Market News: A section displaying cards with the latest exchange rate movements and external links to economic news sites.
Privacy Mode: A settings function that hides converted values (displaying "••••••") for increased discretion.

🎨 FontAwesome (Icons)

faDollarSign: Logo/Currency symbol.
faTimes: The "X" to close the mobile menu.
